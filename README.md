# 🎙️ Realtime AI Assistant

Интеллектуальный помощник для живых собеседований с захватом системного аудио и генерацией ответов на трех языках.

## ✨ Возможности

- 🎤 Захват системного аудио в реальном времени
- 🤖 Транскрипция речи через OpenAI Whisper
- 💬 Генерация ответов на RU, EN, SV языках
- 🔄 Поддержка GPT-4o Realtime моделей
- 📊 Статистика работы в реальном времени

## 🚀 Быстрый старт

1. **Установка зависимостей:**
   ```bash
   pnpm install
   ```

2. **Настройка API ключей:**
   ```bash
   cp .env.example .env.local
   # Отредактируйте .env.local и добавьте ваши API ключи:
   # - OpenAI API ключ для GPT-4o Realtime
   # - Deepgram API ключ для транскрипции
   ```

3. **Запуск приложения:**
   ```bash
   pnpm start
   ```

- **Electron** - Desktop приложение
- **OpenAI Realtime API** - AI модели

- **Express** - Локальный сервер
- **Socket.IO** - WebSocket соединения
