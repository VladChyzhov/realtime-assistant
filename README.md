# Realtime Assistant

Приложение для живого собеседования с использованием OpenAI Realtime API.

## ✅ Статус проекта

**Проект полностью исправлен и готов к работе!** Все критические ошибки устранены.

## 🚀 Быстрый старт

1. **Установите зависимости:**
   ```bash
   pnpm install
   ```

2. **Создайте файл `.env`:**
   ```bash
   # Windows PowerShell:
   Copy-Item env.example .env
   
   # Или вручную создайте файл .env в корне проекта
   ```

3. **Отредактируйте `.env`:**
   ```
   OPENAI_API_KEY=sk-ваш-реальный-ключ-здесь
   ```

4. **Запустите приложение:**
   ```bash
   pnpm start
   ```

## 🎯 Как использовать

1. **Запустите приложение** - откроется Electron окно
2. **Выберите модель** GPT (быстрая или качественная)
3. **Нажмите Start** - приложение подключится к OpenAI
4. **Воспроизводите видео** или **говорите в микрофон**
5. **Смотрите транскрипцию** в реальном времени
6. **Получайте ответы** на трех языках (RU, EN, SV)
7. **Нажмите Stop** для остановки

## 🔧 Устранение неполадок

### ❌ "API ключ не найден"
- Проверьте, что файл `.env` создан в корне проекта
- Убедитесь, что ключ начинается с `sk-`
- Перезапустите приложение после изменения `.env`

### ❌ "Не удалось найти источник экрана"
- Перезапустите приложение
- Убедитесь, что у Electron есть разрешения на доступ к экрану

### ❌ "Ошибка при подключении к OpenAI"
- Проверьте правильность API ключа
- Убедитесь, что у вас есть доступ к Realtime API
- Проверьте интернет-соединение

### ❌ Приложение не запускается
- Убедитесь, что все зависимости установлены: `pnpm install`
- Проверьте, что Node.js версии 18+ установлен
- Попробуйте очистить кэш: `Remove-Item -Recurse -Force node_modules && pnpm install`

## 📋 Возможности

- 🎙️ **Захват аудио с экрана** - транскрибирует любой звук с экрана
- 🗣️ **Распознавание речи в реальном времени** - мгновенная транскрипция
- 🌍 **Генерация ответов на трех языках** - RU, EN, SV
- 🔇 **VAD (Voice Activity Detection)** - оптимизация передачи аудио
- 💻 **Современный Electron интерфейс** - удобное управление
- 🔐 **Автоматическое использование API ключа** из переменных окружения
- 📊 **Детальное логирование** - для отладки и мониторинга

## 🛠️ Технологии

- **Electron 30.5.1** - десктопное приложение
- **OpenAI Realtime API** - распознавание речи и генерация ответов
- **Собственная реализация VAD** - определение речи и тишины
- **WebRTC** - передача аудио в реальном времени
- **ES модули** - современный JavaScript

## 🔒 Безопасность

- **API ключ хранится только** в файле `.env` (не в UI)
- **Файл `.env` не должен попадать** в git репозиторий
- **Приложение автоматически использует** ключ из переменных окружения
- **Нет передачи ключа** через пользовательский интерфейс
