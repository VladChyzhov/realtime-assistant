# ⚡ Быстрая настройка

## 1. Установка зависимостей
```bash
pnpm install
```

## 2. Создание .env файла
```bash
cp env.example .env
```

## 3. Добавление API ключа
Откройте `.env` и замените `sk-your-api-key-here` на ваш ключ OpenAI

## 4. Запуск
```bash
pnpm start
```

## 🎯 Готово!
Приложение откроется и будет готово к работе.
