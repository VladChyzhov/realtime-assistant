# 🚀 Инструкции по настройке Realtime AI Assistant

## 📋 Предварительные требования

- Node.js 18+ 
- pnpm (рекомендуется) или npm
- OpenAI API ключ

## 🔑 Получение OpenAI API ключа

1. Перейдите на [OpenAI Platform](https://platform.openai.com/)
2. Войдите в аккаунт или создайте новый
3. Перейдите в раздел "API Keys"
4. Создайте новый API ключ
5. Скопируйте ключ (начинается с `sk-`)

## ⚙️ Настройка проекта

### 1. Установка зависимостей

```bash
pnpm install
```

### 2. Создание файла конфигурации

```bash
cp env.example .env
```

### 3. Настройка API ключа

Откройте файл `.env` и замените `sk-your-api-key-here` на ваш реальный API ключ:

```env
OPENAI_API_KEY=sk-1234567890abcdef...
MODEL=gpt-4o-realtime-preview
```

### 4. Запуск приложения

```bash
pnpm start
```

- 🤖 Транскрипция речи через OpenAI Whisper

## 🛠️ Устранение неполадок

### Проблема: "Start button не работает"

**Решение:** Проверьте, что файл `.env` создан и содержит правильный API ключ.

### Проблема: "Permission denied" для аудио

**Решение:** Разрешите доступ к микрофону и системному аудио в настройках браузера.

### Проблема: "WebRTC connection failed"

**Решение:** 
1. Проверьте интернет-соединение
2. Убедитесь, что API ключ действителен
3. Проверьте лимиты API OpenAI

## 🔧 Дополнительные настройки

### Переменные окружения

- `PORT` - Порт локального сервера (по умолчанию: 3000)
- `NODE_ENV` - Режим разработки (development/production)
- `STREAMING_TIMEOUT` - Таймаут потока аудио (мс)
- `REALTIME_AUTOCOMMIT_MS` - Интервал авто-коммита (мс)

### Скрипты разработки

```bash
# Режим разработки
pnpm run dev

# Сборка приложения
pnpm run build

# TypeScript компиляция
pnpm run ts
```
